# Get {#form-get}

Inserendo un nome di un form salvato dal form builder, mostra il form stesso nella pagina.

Nome Componente:

```php
livewire:form.get
```

Parametri

```php
string $form_name
```

Esempio:

```php
<livewire:form.get form_name="edit_profile_frontend" />
```
